Book API Documentation

GetBooks GET HEAD

Query Params

  • No supported params

Returns

A JSON array with all the book titles and others.

findByAuthor GET HEAD

Query Params

  • author: string

Returns

A JSON book object.

findByTitle GET HEAD

Query Params

  • title: string

Returns

A JSON book object.

bookCountries GET HEAD

Query Params

  • No supported params

Returns

A JSON array of the countries of the books.

addBook POST

Supported Body Formats

  • application/json
  • application/x-www-form-urlencoded

Body Params

  • author: string
  • country: string
  • link: string (URL)
  • page: number
  • title: string
  • year: number

Returns

A JSON book object representing the new book.

addReview POST

Supported Body Formats

  • application/json
  • application/x-www-form-urlencoded

Body Params

  • title: string
  • rating: number

Returns

A JSON book object representing an updated book, or a 404 if no book with that title exists.
